# War Card Game

This is a simple implementation of the card game "War"

See https://en.wikipedia.org/wiki/War_(card_game)

## Installation

```
npm install
tsc
```

## Usage

For running the program:
```
npm start
```

For running Unit Tests:
```
npm test
```

Tested with npm 8.9.0, node 18.1.0 and TypeScript 4.6.4
